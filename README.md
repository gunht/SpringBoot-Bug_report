## Setup
To run this project, with Docker:
1. Setup Docker
2. Open cmd run:
cd to bug_report folder source code.
```
$ docker build -t bug_report .
```
```
$ docker-compose up -d
```
3. Wait 10 minutes or more. And go http://localhost:8080
